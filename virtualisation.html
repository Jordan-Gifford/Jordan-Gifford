Virtualisation

During my HND, I studied about the basics of Virtualisation. I learnt about the concept and of the different types of hypervisors used within Virtualisation. For example, a type one hypervisor operates on the hardware layer and requires a command prompt to configure the virtual machines but still does have a basic Operating System, which can be accessed vie an external monitor, in order to manage the memory and storage capacity between the hardware and the virtual machines and manage the security of the hypervisor. A good example of a type one hypervisor is Microsoft's Hyper V. Whereas, a type two hypervisor operates above the hardware and alongside the Operating System layer, a good example of one that I have and still use is Virtual box, as Virtual box can be installed and can run on any Operating System platform.
